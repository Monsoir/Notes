# 安装工具

## 安装 Homebrew

```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## 安装 Cocoapods

```sh
sudo gem install cocoapods
```

## 安装 Carthage

```sh
brew install carthage
```

## 安装 Node

```sh
brew install node
```

## 安装 Python3

```sh
brew install python3
```

## 安装 Virtualenv

```sh
pip3 install virtualenv
```
