# Command

### Mac 上通过 Homebrew 安装 MySQL
`brew install mysql`

### 开启与关闭 MySQL 服务
- `mysql.server start`
- `mysql.server stop`

### 使用 root 登陆 MySQL
`mysql -u root -p`

### 查找所有 MySQL 的进程
`$ ps ax | grep mysql`

### 展示所有数据库
`mysql> show databases`

### 选择使用某数据库
`mysql> use <database name>`

### 展示某数据库中的所有表
`mysql> show tables`

### 查看当前连接状态
`mysql> status`

### 默认存储位置
`/usr/local/var/mysql`
