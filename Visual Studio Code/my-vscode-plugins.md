# 我的 VSCode 插件

## A

- Auto Close Tag
    - 自动补全标签的另一半

## B

- Bookmarks
    - 书签标注
- Bracket Pair Colorizer
    - 不同颜色高亮每一对括号，便于区分

## C

- Code Outline
    - 代码结构概览
- Code Runner
    - 快捷运行代码
- Codelf
    - 命名利器
- CSS in JS
    - JS 文件中编写 CSS 时的代码提示

## D

- DotENV
    - .env 文件的语法高亮

## E

- ESLint
    - JS 代码语法及规范检查

## F

- file-size
    - 显示当前编辑文件的文件大小

## I

- Import Cost
    - 显示引入包的大小，针对 JS 文件

## N

- Npm Intellisense
    - npm 包引入代码提示

## P

- Path Intellisense
    - 引入文件时的路径提示
- Polacode
    - 将选中代码生成图片

## R

- React Native Tools
    - React Native 开发工具

## T

- TSLint
    - TypeScript 语法及规范检查

## V

- vscode-icons
    - VSCode 文件图标美化
